# 🎓 Flask Student Result Portal

A full-stack web application designed to manage and display student academic performance records using a modern web architecture.

## 🌟 Key Features
- **Dynamic Grade Logic**: Automated grade assignment (A+ to Fail) based on backend logic.
- **Search Optimization**: Real-time filtering by student name or roll number.
- **Relational Mapping**: Utilizes **Flask-SQLAlchemy** for efficient SQLite database interactions.
- **CRUD Operations**: Complete functionality to Create, Read, and Delete student records.
- **Responsive Design**: Built with **Bootstrap 5** for a professional look on all devices.

## 🛠️ Tech Stack
- **Backend**: Python 3.13, Flask
- **Database**: SQLite & SQLAlchemy (ORM)
- **Frontend**: HTML5, Jinja2 Templates, Bootstrap 5
- **Language**: Python
